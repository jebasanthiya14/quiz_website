# quiz_website
Sports Trivia Challenge
A fun and interactive Sports Trivia Game built using HTML, CSS, and JavaScript.
This project allows users to test their knowledge across multiple sports categories including Football, Basketball, Tennis, and the Olympics.

Features
Multiple sports categories to choose from
Dynamic questions and answer validation
Progress tracker displaying the current question number
Final scoreboard showing total score after the quiz
Option to restart and play again
Simple and responsive user interface

Technologies Used
HTML5 – for structure and layout
CSS3 – for design and styling
JavaScript – for game logic and interactivity
